9/14 & 15Conference 

On September 14th and 15th, you'll join 1,400 engineers, technical leads, open source developers, and business leaders at GitHub Universe. The event features two keynotes, six featured speakers, and twenty-four breakout sessions that dive deep into technical and team challenges of building software.

Together we'll explore what it takes to define the future of software. Hear how individuals and teams are using modern tools, processes, and cultures to tackle the ever-bigger challenges we face. We'll look at ways to apply that software to impact everything from communities to businesses.

Software is more than code and at Universe we explore it all.

9:00 AM
Getting Started with InnerSource

Are you eager to apply the successful principles of open source to your internal projects? In this session, we will discuss a practical roadmap for moving your organization toward a more successful InnerSource model.

The Plumbers Guide to the Galaxy: Implementing a CI/CD Pipeline with GitHub

GitHub makes it easy to collaborate on code, but its true power comes when it is used as the centerpiece of a modern deployment pipeline. This workshop will start with an overview of Git and GitHub, then deep dive into the concepts behind Continuous Integration (CI) and Continuous Delivery (CD), two cornerstones of good software development. After that, we'll cover how to start incorporating these practices into your workflow, by using GitHub to integrate with best-of-breed 3rd party tools.

12:00 PM

Lunch

1:30 PM
Blasting off with the GitHub API

GitHub's powerful API is for more than integrations. In this hands on session, you will learn how to create several practical scripts that leverage the GitHub API to supercharge your workflows.

GitHub Integrations & the Project Lifecycle

Learn more about how to improve your code with GitHub integrations. We'll explore tools that work seamlessly with your pull requests to help you ship better quality code.

3:00 PM

Break

3:30 PM
Git Cozy

Git is an incredibly powerful tool but it can be difficult to learn. In this session, we will use visualization tools and examples of advanced scenarios to help you form a mental model of the Git workflow. You will leave with the confidence and composure to 'git' out of your next sticky situation.

Life Embetterment with Hubot

Just as DevOps is about knocking down barriers between developers and operations, ChatOps helps teams embrace and extend this philosophy by putting tools in the middle of the conversation. In this session you'll learn how to use Hubot, GitHub's "life embetterment robot," to improve your organization's culture of automation, measurement, and sharing.

Platform Day

Integrators, partners, and other teams building on-top of the GitHub API have an opportunity to come together before Universe. This mini-conference brings together the GitHub ecosystem to learn from each other, share best practices, and hear from our Platform and Business Development teams. The event is open by invitation to people facing the unique challenges of building on the GitHub platform.

Request an invitation to platform day

Ask Services

Returning to Universe is the opportunity to get one-on-one time with our Field Services team. Designed for businesses both small and large, this is a complimentary 30-minute consultation to discuss workflows, use cases, migrations, configurations of your GitHub.com or GitHub Enterprise instance, or any other Git topics you want to tackle. Ask Services is available to all Universe attendees but due to limited availability you must reserve your spot in advance.

After you've registered for Universe, reserve your Ask Services session.
